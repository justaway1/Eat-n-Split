## Installation
1. Clone the repository: 
    ```bash
    git clone https://github.com/justaway1/Eat-n-Split.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Eat-n-Split
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
